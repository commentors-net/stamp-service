Secure Stamp Service has been successfully installed and started.

NEXT STEPS (CRITICAL):

1. CREATE BACKUP SHARES

   If you selected the option, an Admin CLI window will open.
   Run this command:

   StampService.AdminCLI.exe create-shares --total 5 --threshold 3 --output C:\Shares

   This creates 5 shares, requiring any 3 to recover the key.

2. DISTRIBUTE SHARES SECURELY

   • Give each share to a different trusted person
   • Store shares offline (USB drive, paper printout, safe)
   • Each person should verify their share works
   • Document who has which share
   • NEVER store all shares together!

3. VERIFY INSTALLATION

   Open Command Prompt or PowerShell and run:

   sc query SecureStampService
   cd "C:\Program Files\StampService\AdminCLI"
   StampService.AdminCLI.exe status
   StampService.AdminCLI.exe test-stamp

   You should see the service running and tests passing.

4. SAVE PUBLIC KEY

   The public key can be safely shared with client applications:

 StampService.AdminCLI.exe status > public-key.txt
