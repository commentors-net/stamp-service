# ?? AdminGUI Polish Features - Quick Reference

## ?? Keyboard Shortcuts

### **Press F1 anywhere** for full help!

| Shortcut | Action |
|----------|--------|
| **F1** | Show keyboard shortcuts help |
| **F5** | Refresh |
| **Ctrl+R** | Refresh |
| **Ctrl+F** | Focus search |
| **Escape** | Close dialog |

### **Dashboard**
- `Ctrl+N` - Create Token
- `Ctrl+M` - Manage Secrets
- `Ctrl+B` - Backup & Recovery
- `Ctrl+H` - Service Health

### **Secret Manager**
- `Delete` - Delete selected
- `Enter` - View details
